instructions:

NOTE: you must have installed Blackhole, sox, and Spotipy for this to run properly. here's the commands:
 
brew install --cask blackhole-2ch
brew install sox
pip install spotipy 


0) open spotify

1) make sure all other apps are either closed or muted 
1b) (you may want to send your notification sounds to another interface eg. MacBook speakers)

2) enter your client id, secret, and playlist id in the fields for spotisplit.py

3) type "%run spotisplit.py" in terminal
3b) WAIT for duration of playlist (don't play any sounds or change the volume!!)

4) songs will be output into a folder with the title of the playlist in .wav format
